# Katsuo

A sample that adopted the latest trend in Android development

## Architecture

- Multi modules
- MVVM

## Libraries

- Android Jetpack
  - Data binding
  - Lifecycle
  - ViewModel
  - LiveData
  - KTX
  - Room
- OkHttp
- Retrofit
  - Kotlin coroutines adapter
  - KotlinX serialization converter
- Glide
